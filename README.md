# 숫자 야구 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 구현할 기능 목록
* 데이터 입력 및 메시지 출력
* 초기 정답 랜덤화
* 각 자릿수를 비교하여 스트라이크 및 볼 여부 판단 및 출력
* 오답 혹은 정답 시 루프 및 종료 로직 구현
